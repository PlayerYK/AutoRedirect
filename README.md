autoRedirect
============

##What is this repository for?

Chrome extension to replace current url
使用场景：本地环境与dev环境自动切换。

* 在浏览器中打开的文件或者网址，自动重新定位到新的网址。
* 如果匹配到多个结果，会提示你选择你想要打开的网址。
